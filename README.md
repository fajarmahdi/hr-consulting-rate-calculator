Overview
Interactive calculator for HR consulting pricing strategies. Converts client revenue/profit data into four pricing models with real-time calculations. Built as single-file HTML for instant deployment.

Features
Live calculations across four pricing options (Pure Contract, Cash+Credit, Variable/Visit, Headcount Scaler)
Sequential input unlocking - fields activate based on prior entries
Smart anchoring prioritizes GM Salary > Profit > Revenue for base rates
Profit capping at 10% GOP maximum
IDR formatting with billions/millions auto-scaling
Responsive design using Inter font from Google CDN
No backend required - pure client-side JavaScript

Input Flow
Monthly Revenue (0.4% base rate if no better data)
GOP (1.5% base rate, caps at 10%)
GM Salary (60% target anchor)
Headcount (enables Option D scaling)
Visit Days (Option C variable pricing)
